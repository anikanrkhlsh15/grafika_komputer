## Cara Menggunakan

1. Buka file `index.html` di browser
2. Izinkan akses kamera ketika diminta
3. Pilih grid yang diinginkan
4. Tersenyum atau buat ekspresi wajah untuk melihat efek emoji
5. Klik "Ambil Foto" untuk mengambil gambar
6. Gunakan "Unduh Foto" untuk menyimpan foto
7. "Reset" untuk membersihkan galeri

## Teknologi

- ML5.js dengan Facemesh untuk deteksi wajah
